# Dashboard Mundial

Dashboard web desarrollado con Flask que consume datos de la API del Banco Mundial para visualizar indicadores por país.

## Funcionalidades

- Selección de país.
- Comparación entre dos países.
- Selección de indicador para el gráfico:
  - Población total
  - PIB
  - Esperanza de vida
- KPIs principales:
  - Población actual
  - PIB actual
  - Esperanza de vida actual
  - Crecimiento poblacional
- Gráfico dinámico con Chart.js.
- Tabla histórica del indicador seleccionado.

## Tecnologías utilizadas

- Python
- Flask
- Pandas
- Requests
- HTML
- CSS
- Chart.js
- API Banco Mundial

## Objetivo del proyecto

El objetivo fue construir un dashboard interactivo aplicando consumo de APIs, transformación de datos con Pandas y visualización web con Flask y Chart.js.
